# ness.OS + ness.WEB

Ecossistema NESS: plataforma interna (ness.OS) + site público (ness.WEB) em uma única base Next.js.

- **Site público:** Blog, Soluções, Carreiras, Casos, Contato, Chatbot RAG
- **App interno (ness.OS):** Growth, OPS, FIN, PEOPLE, JUR, GOV

**Stack:** Next.js 14 (App Router), Supabase, Tailwind, shadcn/ui, Vercel AI SDK, pgvector
